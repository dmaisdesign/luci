Política de Privacidade

Última atualização: 05 de março de 2026

A sua privacidade é importante para nós. Esta Política de Privacidade descreve como o site LucidBag, acessível em https://lucidbag.com.br/, coleta, utiliza, armazena e protege as informações dos usuários.

Ao acessar e utilizar nosso site, você concorda com os termos descritos nesta Política de Privacidade.

1. Informações que coletamos

Podemos coletar diferentes tipos de informações quando você acessa ou interage com o nosso site, incluindo:

1.1 Informações fornecidas pelo usuário

Podemos coletar dados que você fornece voluntariamente, como:

  • Nome

  • Endereço de e-mail

  • Informações enviadas através de formulários

  • Mensagens enviadas pelo formulário de contato

1.2 Informações coletadas automaticamente

Quando você acessa nosso site, algumas informações podem ser coletadas automaticamente, incluindo:

  • Endereço IP

  • Tipo de navegador

  • Dispositivo utilizado

  • Páginas acessadas

  • Tempo de navegação

  • Origem do acesso

Esses dados são utilizados para melhorar a experiência do usuário e o funcionamento do site.

2. Uso das informações

As informações coletadas podem ser utilizadas para:

  • Melhorar o conteúdo e a experiência do site

  • Responder solicitações ou mensagens enviadas pelos usuários

  • Personalizar a navegação

  • Monitorar desempenho e segurança do site

  • Prevenir fraudes ou atividades maliciosas

3. Cookies

O site LucidBag pode utilizar cookies para melhorar a experiência de navegação.

Cookies são pequenos arquivos armazenados no navegador do usuário que ajudam a:

  • lembrar preferências

  • analisar o tráfego do site

  • melhorar funcionalidades

Você pode desativar os cookies nas configurações do seu navegador. No entanto, algumas funcionalidades do site podem ser afetadas.

4. Google Analytics e ferramentas de terceiros

Podemos utilizar ferramentas de terceiros para análise de tráfego e desempenho do site, como:

  • Google Analytics

  • ferramentas de publicidade

  • serviços de monitoramento

Esses serviços podem coletar dados anônimos de navegação para gerar relatórios estatísticos.

5. Compartilhamento de informações

O site LucidBag não vende, aluga ou comercializa dados pessoais dos usuários.

As informações podem ser compartilhadas apenas quando necessário:

  • para cumprir obrigações legais

  • para proteger direitos legais

  • para garantir a segurança do site

6. Links para sites externos

Nosso site pode conter links para sites externos. Não somos responsáveis pelas políticas de privacidade ou conteúdo desses sites.

Recomendamos que o usuário leia as políticas de privacidade de cada site visitado.

7. Segurança das informações

Adotamos medidas técnicas e organizacionais para proteger as informações dos usuários contra:

  • acesso não autorizado

  • alteração

  • divulgação

  • destruição de dados

No entanto, nenhum sistema de transmissão ou armazenamento de dados é completamente seguro.

8. Direitos do usuário (LGPD)

De acordo com a Lei Geral de Proteção de Dados (LGPD), o usuário tem direito a:

  • acessar seus dados pessoais

  • solicitar correção de dados incorretos

  • solicitar exclusão de dados

  • revogar consentimentos

  • obter informações sobre uso de dados

Caso deseje exercer algum desses direitos, entre em contato conosco.

9. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada periodicamente para refletir mudanças no site ou na legislação.

Recomendamos que os usuários revisem esta página regularmente.

10. Contato

Se você tiver dúvidas sobre esta Política de Privacidade ou sobre o tratamento de dados, entre em contato através do site:

https://lucidbag.com.br/
